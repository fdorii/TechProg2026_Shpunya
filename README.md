## 1 Задача на выполнение - <code>Сервер несколько клиентов</code>

Для выполнения этого задания в исходной конфигурации сервера в файле <code>mytcpserver.cpp</code> и <code>mytcpserver.h</code> были выполнены некоторые изменения:
1. В .h:

 - Убран одиночный <code>mTcpSocket *</code>
 - Добавлен <code>QList<QTcpSocket*> mClients</code> — список всех подключённых клиентов
 - Добавлен <code>QMap<QTcpSocket*, QString> mBuffers</code> — отдельный буфер данных для каждого клиента

2. В .cpp:

 - <code>slotNewConnection()</code> — каждый новый сокет добавляется в список, сигналы подключаются к тем же слотам
 - <code>slotServerRead() и slotClientDisconnected()</code> — используют <code>sender()</code>, чтобы определить, от какого именно клиента пришёл сигнал. Это ключевой момент для многопользовательности
 - При отключении клиент удаляется из списка и буфера, вызывается <code>deleteLater()</code> для безопасного освобождения памяти
